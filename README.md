# HR TCC Service

Node.js сервис для HR системы.

## Установка

```bash
npm install
```

## Запуск

Для разработки:
```bash
npm run dev
```

Для продакшена:
```bash
npm start
```

## Тестирование

```bash
npm test
```

## Структура проекта

```
src/
  ├── index.js      # Точка входа приложения
  ├── routes/       # Маршруты API
  ├── controllers/  # Контроллеры
  ├── models/       # Модели данных
  └── middleware/   # Промежуточное ПО
```
